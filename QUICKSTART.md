# 🚀 快速启动指南

## 一键启动（推荐）

```bash
# 在项目根目录执行
./start-services.sh
```

然后打开浏览器访问：http://localhost:8000/video_player.html

---

## 手动启动

### 步骤 1：启动后端服务

打开终端 1：
```bash
cd backend
node server.js
```

看到以下输出表示成功：
```
✅ 使用阿里云百炼（通义千问）API
🚀 服务器运行在 http://localhost:3000
📡 API 端点: http://localhost:3000/api
```

### 步骤 2：启动前端服务

打开终端 2（保持终端 1 运行）：
```bash
# 在项目根目录
python3 -m http.server 8000
```

看到以下输出表示成功：
```
Serving HTTP on :: port 8000 (http://[::]:8000/) ...
```

### 步骤 3：访问应用

打开浏览器访问：
- **主页面**：http://localhost:8000/video_player.html

## 测试 AI 朋友圈功能

### 1. 打开 AI 朋友圈
- 点击视频播放器右下角的 **👥** 按钮
- 侧边栏从右侧滑出

### 2. 发布动态
1. 点击"✏️ 发布动态"按钮
2. 输入内容，例如："今天看了这部剧，真的太好看了！"
3. 点击"发布"
4. **等待 10-30 秒，AI 会自动点赞或评论**

### 3. 发表评论
1. 在任意动态下方的输入框输入评论
2. 点击"发送"
3. **等待 3-8 秒，AI 会自动回复**

### 4. 验证 AI 是否工作

✅ **正常工作的标志：**
- 发布动态后，10-30 秒内有 AI 互动
- 发表评论后，3-8 秒内有 AI 回复
- 后端终端显示"✅ AI 回复成功"
- 页面每 5 秒自动刷新

## 常见问题

### Q1: 端口被占用怎么办？

```bash
# 查找占用端口的进程
lsof -ti:3000

# 终止进程（替换 <PID> 为实际进程号）
kill -9 <PID>
```

### Q2: AI 没有回复怎么办？

1. 检查后端终端是否有错误信息
2. 确认 API Key 配置正确（backend/.env）
3. 查看浏览器控制台（F12）是否有错误
4. 等待更长时间（最多 30 秒）

### Q3: 前端无法连接后端？

1. 确认后端服务已启动（终端 1 有输出）
2. 访问 http://localhost:3000/api/moments 测试 API
3. 检查浏览器控制台的网络请求

### Q4: 如何停止服务？

```bash
# 使用停止脚本
./stop-services.sh

# 或者在两个终端中按 Ctrl+C
```
