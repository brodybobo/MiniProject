# 视频播放器 WebUI

一个仿腾讯视频风格的本地视频播放器界面，支持本地视频文件播放。

## 功能特性

### ✅ 已实现功能

1. **视频播放**
   - 支持本地视频文件上传播放
   - 支持拖拽上传视频
   - 基于 Video.js 的专业播放器
   - 播放、暂停、进度控制
   - 音量调节
   - 全屏播放
   - 倍速播放（0.5x - 2x）

2. **界面布局**
   - 顶部导航栏（Logo、菜单、搜索、用户中心）
   - 左侧主视频播放区域
   - 右侧信息栏（剧集信息、选集面板、推荐视频）
   - 响应式设计，支持多种屏幕尺寸

3. **选集功能**
   - 40集剧集列表展示
   - 点击切换集数
   - 当前集高亮显示
   - 播放结束自动下一集

4. **键盘快捷键**
   - `空格键`: 播放/暂停
   - `←/→`: 后退/前进 5秒
   - `↑/↓`: 增加/减少音量
   - `F键`: 全屏切换

5. **其他功能**
   - 剧集信息展示
   - 演员表展示
   - 推荐视频列表
   - 操作按钮（追剧、提醒、分享）

## 使用说明

### 1. 打开页面

直接在浏览器中打开 `video_player.html` 文件即可。

**推荐浏览器：**
- Google Chrome
- Microsoft Edge
- Firefox
- Safari

### 2. 播放视频

有两种方式加载视频：

**方法一：点击上传按钮**
1. 点击视频播放器中央的"选择视频文件"按钮
2. 在文件选择对话框中选择本地视频文件
3. 视频将自动加载到播放器中

**方法二：拖拽上传**
1. 直接将视频文件拖拽到视频播放器区域
2. 释放鼠标，视频将自动加载

**支持的视频格式：**
- MP4 (推荐)
- WebM
- OGG
- MOV
- AVI
- MKV（部分浏览器支持）

### 3. 播放控制

**鼠标操作：**
- 点击播放器中央或底部播放按钮：播放/暂停
- 拖动进度条：跳转到指定位置
- 点击音量图标：静音/取消静音
- 拖动音量滑块：调节音量
- 点击全屏按钮：进入/退出全屏

**键盘快捷键：**
- `空格`: 播放/暂停
- `←`: 后退5秒
- `→`: 前进5秒
- `↑`: 音量+10%
- `↓`: 音量-10%
- `F`: 全屏切换

### 4. 切换集数

- 在右侧"选集面板"中点击集数按钮
- 当前播放集会高亮显示
- 播放结束后会自动切换到下一集

### 5. AI朋友圈功能

- **打开朋友圈**: 点击播放控制栏的👥按钮打开AI朋友圈侧边栏
- **发布动态**: 点击"发布动态"按钮分享观剧感受
- **查看动态**: 浏览AI好友的评论和互动
- **点赞评论**: 对喜欢的动态进行点赞和评论
- **关闭侧边栏**: 点击右上角X按钮或按ESC键关闭

### 6. 其他功能

- **追剧按钮**: 点击可添加到追剧列表（演示功能）
- **提醒按钮**: 设置更新提醒（演示功能）
- **分享按钮**: 分享功能（演示功能）
- **推荐视频**: 点击可查看推荐内容（演示功能）

## 自定义修改

### 修改剧集数量

在 `script.js` 中修改：
```javascript
const totalEpisodes = 40; // 修改为你需要的集数
```

### 修改剧集标题

在 `script.js` 的 `switchEpisode` 函数中修改：
```javascript
videoTitle.textContent = `你的剧集名 第${episodeText}集`;
```

### 修改主题颜色

在 `styles.css` 中搜索 `#ff6b00`（主题橙色）并替换为你喜欢的颜色。

### 添加更多推荐视频

在 `video_player.html` 的 `.recommend-list` 部分添加更多 `.recommend-item` 元素。

### 配置AI朋友圈

#### 修改AI好友数据
在 `ai-moments-data.js` 中修改AI好友信息：
```javascript
const aiFriends = [
    {
        id: 1,
        name: "AI好友名称",
        avatar: "👤",
        personality: "性格描述"
    }
];
```

#### 自定义动态内容
在 `ai-moments.js` 中修改AI生成动态的模板：
```javascript
const momentTemplates = [
    "我正在观看《剧集名》，{comment}",
    "第{episode}集太精彩了！{comment}"
];
```

#### 启用/禁用AI朋友圈
在 `script.js` 中注释掉AI朋友圈初始化代码：
```javascript
// 注释此行可禁用AI朋友圈功能
initAIMoments();
```

## 常见问题

**Q: 为什么视频无法播放？**
A: 请确保：
1. 视频格式被浏览器支持（推荐使用MP4）
2. 视频文件没有损坏
3. 浏览器已允许自动播放权限

**Q: 为什么页面样式显示不正常？**
A: 请检查：
1. 确保 `styles.css` 文件在同一目录下
2. 确保有网络连接（加载 Video.js CDN）

**Q: 如何离线使用？**
A: 下载 Video.js 到本地：
1. 访问 https://videojs.com/ 下载最新版本
2. 修改 HTML 中的 CDN 链接为本地路径

**Q: 能否添加字幕功能？**
A: Video.js 支持字幕，可以参考官方文档添加 WebVTT 字幕文件。

**Q: AI朋友圈功能如何工作？**
A: AI朋友圈是一个模拟社交功能，通过预设的AI好友数据和智能模板生成动态内容，提供社交化的观看体验。

**Q: 如何自定义AI好友？**
A: 在 `ai-moments-data.js` 文件中修改AI好友的姓名、头像和性格设置。

**Q: 能否禁用AI朋友圈功能？**
A: 可以，在 `script.js` 中注释掉 `initAIMoments()` 调用即可禁用该功能。

**Q: AI朋友圈数据会保存吗？**
A: 当前版本中，AI朋友圈数据仅在当前会话中保存，刷新页面后会重置。生产环境可集成后端服务实现数据持久化。
